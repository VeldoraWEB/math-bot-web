# Чат-бот "Math-Bot"

Простой чат-бот для выполнения арифметических операций над числами и запоминание юзера. Работает полностью в браузере.

## Как запустить

Перейти на GithubPages https://veldoraweb.github.io/math-bot-web/

## Команды
| Команда | Действие |
|---------|----------|
| `/start` | Начать общение, представиться |
| `/name: Имя` | Указать имя (после /start) |
| `/number: 7 9 134` или `/n: 7 9 134` | Ввести числа для вычисления |
| `+` `-` `*` `/` | Выбрать действие |
| `/stop` | Закончить диалог |

## Особенности
- Поддерживаются целые и дробные числа
- Разделители: пробелы, запятые, точки с запятой
- После одного вычисления можно сразу вводить новые числа
- Адаптивный дизайн под все устройства (макс. ширина 700px)
- Анимация печати (троеточие) при ответе бота
